> Onegai My Kuromi is a 2d platformer game inspired by Kuromi from San Rio.

## Installation

Install my-project with Git

```bash
  git clone https://github.com/zielww/Onegai-My-Kuromi.git
  cd Onegai-My-Kuromi

```

Install pygame with pip

```bash
  pip install pygame
```
